    Operators

    This local backend keeps the repo defaults. Additional operator selection and partner platform integration are not wired yet.

    Host discovery

    Host discovery is handled by the existing stakkr agent ISO flow. The plugin renders the local-only input files, generates the agent image through the site playbook, and lets the cluster discovery happen through that path.

    Storage

    Compact uses the existing raw logical-volume path from /dev/ocptb. SNO uses the current repo VM defaults. Storage sizing is not configurable in this first backend pass.
