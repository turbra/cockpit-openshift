    Networking guidance

    • Reserve stable private IPs for each control plane host.
    • The generated template assigns that static IP on the primary NIC unless a private VLAN ID is provided.
    • Add a VLAN subinterface when the cluster network lives on a tagged network.
    • Switch to YAML view when you need bonds, multiple interfaces, or custom routes.
